# **App Name**: Dharani's Digital Domain

## Core Features:

- Personal Information Display: Display personal details: name, date of birth, address, email, and phone.
- Professional Summary: Showcase professional background: education, skills, certifications, and experience.
- Skills Section: List key skills, including Security & Incident Response, Platforms, and Programming Languages.
- Projects Portfolio: Exhibit personal projects, with descriptions and links to the respective GitHub repositories.
- CV Download: Ability to download a PDF version of the CV.
- Contact Form: A contact form to allow visitors to send messages directly.
- 3D Element Integration: Incorporate subtle 3D elements and animations to enhance the user experience and visual appeal.
- Advanced Tech Design: Implement a tech-forward design with interactive components and a dynamic layout.

## Style Guidelines:

- Primary color: Dark navy blue (#2E3148) to convey professionalism and stability.
- Background color: Light grey (#F0F0F0) for a clean, modern look.
- Accent color: Soft teal (#66A5AD) for highlights and interactive elements, adding a touch of sophistication.
- Body font: 'Inter', sans-serif, for a modern and readable feel in longer text blocks.
- Headline font: 'Space Grotesk', sans-serif, for headlines, conveying a techy, modern, scientific impression.
- Use minimalist, professional icons for skills and certifications. Should use font-awesome
- Employ a clean, grid-based layout for easy navigation. Use clear section headings to separate content.
- Subtle transitions and animations to guide the user through the content and highlight important sections.
